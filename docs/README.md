# FAIR CRM Documentation

Bu projede ana dokümantasyon ve proje takibi artık tek Excel dosyasında tutulur:

```text
FAIR_CRM_PROJECT.xlsx
```

Önce bu dosyayı açın. İçindeki **Sekme Rehberi** sayfası hangi sekmenin ne işe yaradığını açıklar.

Eski `.md` dokümanları bilgi kaybı olmaması için silinmedi; şu klasöre arşivlendi:

```text
docs/archive_md_reference/
```

## Ana takip dosyası

| Dosya | Amaç |
|---|---|
| `FAIR_CRM_PROJECT.xlsx` | Roadmap, proje durumu, database dictionary, API listesi, import mapping, testler, kararlar ve changelog |
| `MASTER_CONTEXT.md` | Yeni sohbette/proje devamında okunacak ana bağlam |
| `README.md` | Kurulum ve hızlı başlangıç |
| `CHANGELOG.md` | Sürüm geçmişi |
| `RELEASE_NOTES.md` | Son kullanıcıya yönelik sürüm notları |
| `PROJECT_STATUS.md` | Kısa proje durum özeti |
